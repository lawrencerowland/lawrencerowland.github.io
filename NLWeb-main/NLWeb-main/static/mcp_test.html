About NLWeb and MCP

NLWeb aims to bring AI-powered conversational search to a broad range of content sources, especially websites. NLWeb is designed to give agency into all players in the AI ecosystem, especially content creators. A more balanced ecosystem is one where content creators have greater control over their content.

MCP (Model Context Protocol) is an emerging protocol for Chatbots and AI assistants to interact with tools. Every NLWeb instance is also an MCP server, which supports one method, ask, which is used to ask a website a question in natural language. The returned response is in schema.org, a widely-used vocabulary for describing web data.

This test interface allows you to experiment with MCP requests to your NLWeb server, testing both standard and streaming responses.

Request Format

An MCP request includes:

  • function_call: The function to invoke
  • name: ask
  • arguments: JSON string with parameters. All parameters except 'query' and 'site' are optional. Site may be 'all' to search all the sites in this NLWeb instance.

Response Format

Standard responses include:

  • type: "function_response"
  • status: "success" or "error"
  • response: The answer from the NLWeb instance, in schema.org format

Streaming responses include:

  • type: "function_stream_event" for chunks, "function_stream_end" for completion
  • content: Contains partial_response for each chunk
  • status: Final status in the end event

This test interface provides:

  • JSON Tree Viewer: Interactive expandable/collapsible tree view for JSON responses
  • Resizable Panels: Drag the divider to resize the left and right panels
  • Standard Requests: Test regular JSON requests
  • Streaming Requests: Test Server-Sent Events (SSE) streaming
  • Examples: Pre-configured examples for quick testing
  • Visual Feedback: See response content and status in real-time

Tips

  • Switch between Tree View and Raw JSON/Text modes using the buttons above the response area
  • Click on the toggles (+ / -) in the tree view to expand or collapse JSON nodes
  • For local testing, ensure CORS is properly configured on your server
  • The streaming tab formats partial responses as they arrive
